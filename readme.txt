必读：
1，先手动从屏幕最左上角截图两次：
第一次刚好不包含文字框；第二次刚好包含全部可能范围的文字框
2，右键属性查看截图的分辨率坐标
3，用vscode打开整个文件夹
4，修改jietu.py中的x，y，x1，y1
5，自己申请好百度图片翻译的api，在api.py中填入账密
6，安装好必要的库后运行q.py，每次按q键都会截图并翻译一次，30s后自动关闭以释放内存

建议：
#在游戏窗口最大化后截图运行，这样不会挡住游戏内容
#该程序以2560*1600的分辨率200%缩放比例为基准设计，可以根据自己的显示器修改windows.py中字幕窗口大小
